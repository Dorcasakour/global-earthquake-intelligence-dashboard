# Global Earthquake Intelligence Dashboard

## Project Overview

This project analyzes more than 35,000 earthquake events recorded globally between 2000 and 2025. Using Power BI, the dashboard transforms seismic records into interactive visualizations that support geographic risk assessment, trend monitoring, and earthquake severity analysis.

## Objectives

* Analyze global earthquake occurrence patterns.
* Identify high-risk countries and regions.
* Examine earthquake magnitude and depth relationships.
* Explore temporal trends in seismic activity.
* Support disaster risk awareness through data visualization.

## Dataset Summary

* Total Earthquakes: 35,100+
* Time Period: 2000–2025
* Maximum Magnitude: 9.1
* Average Magnitude: 5.35
* Average Depth: 54.52 km

## Key Findings

* Over 35,000 earthquake events were recorded during the study period.
* Earthquake activity was highly concentrated along major tectonic plate boundaries.
* Indonesia recorded the highest number of earthquake events among analyzed countries.
* Moderate-magnitude earthquakes accounted for the majority of recorded events.
* Significant seismic clusters were observed around the Pacific Ring of Fire.
