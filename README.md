# 🌍 Global Earthquake Intelligence Dashboard  
### Analyzing Seismic Activity and Risk Patterns (2000–2025)

---

## 📊 Project Overview

The Global Earthquake Intelligence Dashboard is an interactive Power BI project designed to analyze and visualize over **35,000+ earthquake events recorded globally between 2000 and 2025**.

The dashboard transforms raw seismic data into meaningful insights that support **geospatial risk analysis, trend monitoring, and disaster awareness**.

This project demonstrates how data analytics and business intelligence tools can be applied to real-world environmental challenges.

---

## 🎯 Business Objective

The main objectives of this project are to:

- Identify global earthquake occurrence patterns  
- Detect high-risk countries and seismic zones  
- Analyze relationships between magnitude and depth  
- Explore long-term seismic trends (2000–2025)  
- Support disaster risk awareness through data visualization  

---

## 📦 Dataset Summary

- **Total Earthquakes:** 35,100+  
- **Time Period:** 2000–2025  
- **Maximum Magnitude:** 9.1  
- **Average Magnitude:** 5.35  
- **Average Depth:** 54.52 km  

---

## 🧠 Key Insights

- Earthquake activity is heavily concentrated along tectonic plate boundaries  
- The Pacific Ring of Fire is the most active seismic region globally  
- Indonesia recorded the highest number of earthquake events  
- Most earthquakes fall within the moderate magnitude range  
- No strong linear relationship was observed between depth and magnitude
